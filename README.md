## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Local Development

```bash
# Clone the repository
git clone https://github.com/premtsd-code/premtsd-code.github.io.git

# Navigate to project directory
cd premtsd-code.github.io

# Install dependencies
npm install

# Start development server
npm start
```

The application will be available at `http://localhost:4200/`

## 📦 Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm run build:gh-pages` - Build for GitHub Pages with correct base href
- `npm run deploy` - Build and deploy to GitHub Pages manually

## 🌐 GitHub Pages Deployment

### Automatic Deployment (Recommended)

The portfolio automatically deploys to GitHub Pages when you push to the main branch using GitHub Actions.

```bash
# Make changes and commit
git add .
git commit -m "Update portfolio"
git push origin main
```

### Manual Deployment

```bash
# Install dependencies
npm install

# Deploy to GitHub Pages
npm run deploy
```

## ⚙️ GitHub Pages Setup

1. **Repository Settings**: 
   - Go to your GitHub repository settings
   - Navigate to "Pages" section
   - Set source to "GitHub Actions"
   - Save settings

2. **Domain Configuration**:
   - Your site will be available at: `https://premtsd-code.github.io/`
   - Custom domains can be configured in repository settings

To customize this portfolio for your own use:

2. **Styling**: Modify `src/app/app.component.css`
3. **Content**: Edit `src/app/app.component.html`
4. **Assets**: Replace files in `public/` folder
5. **Deployment**: Update repository name in scripts

## 🔧 Troubleshooting

### Common Issues

1. **Build Errors**: Ensure Node.js 18+ is installed
2. **Deployment Fails**: Check GitHub Actions permissions
3. **Routing Issues**: Verify 404.html is in public folder
4. **Styling Problems**: Clear browser cache and rebuild
